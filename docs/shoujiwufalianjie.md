# 为什么连接不成功？（手机用户）

### 【苹果iOS用户】

常见解决方案：[👉使用教程点此查看](https://www.proxy-support.com/?p=33)

\*需要确保您的小火箭提前准备好**第三方的海外加速节点(需要自行网速搜索)**，在代理通过选择一个第三方的海外加速节点

\*如果小火箭代理**检测超时先不理会**，直接打开手机浏览器访问：ipapi.co或ip-api.com看看ip是否能对应上&#x20;

\*为了避免手填代理信息出现错误，您可以选择扫码导入代理信息&#x20;

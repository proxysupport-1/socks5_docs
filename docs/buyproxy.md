# 如何选择合适的代理

**一般养号我们会更加推荐静态代理**

**静态代理**有IPV4，IPV6以及静态住宅ISP这三种类型，**其中IPV4和IPV6都是机房代理**，这两种类型的代理一般用于对IP要求不高的平台。<mark style="color:red;">**特别需要注意的是IPV6是近几年新出的代理IP类型，有很多平台暂时都不支持访问，例如tiktok，outlook等都无法打开**</mark>，如果您要购买IPV6之前请领取试用或者咨询人工客服了解。**如果对于一些IP要求更高的平台（社媒，电商类等）会更加建议您选择静态住宅**，其中带有pro标识的专区也是质量更好的代理专区。

**动态代理**的选择更多是用于**注册类**，或者是代理**IP需要经常变化**的业务场景。动态相比静态的好处就是灵活，只要您有流量就能生成代理，以及也可以选择相对的州，城市类的去生成IP。

&#x20;

动态和静态不同的区别就是：

* 动态：代理IP会发生变化，按照流量收费，有流量就可以无限生成代理IP
* 静态：静态IP是固定不会发生变化，流量不限，速度在2到10mbps这个区间，如需更高的速度也可联系在线客服。

&#x20;

> **总结：养号类选静态住宅，对于IP有更高要求的选静态住宅pro专区的，如果对于IP没更高要求的可选IPV4。注册类或者是业务场景需要IP有变化的则选动态。**
